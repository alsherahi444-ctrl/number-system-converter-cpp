# Number System Converter (C++)

A console-based number system converter that supports:

- Binary (Base 2)
- Octal (Base 8)
- Decimal (Base 10)
- Hexadecimal (Base 16)

## Features
- Convert between any of the supported bases
- Input validation
- Clean modular functions

## Example
Input:
Source Base: 2
Target Base: 10
Number: 1011

Output:
11
